# Research Notes

Salvare qui note di ricerca riutilizzabili, verifiche lunghe e confronti tra
fonti.

Convenzione nome file:

```text
YYYY-MM-DD-slug.md
```

Ogni nota dovrebbe indicare:

- leggenda o luogo;
- domanda di ricerca;
- fonti consultate con data;
- claim supportati;
- conflitti o caveat;
- decisione editoriale consigliata.
