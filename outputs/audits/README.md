# Audits

Questa cartella raccoglie audit visuali, editoriali, workflow e QA speciali.

Ogni audit dovrebbe indicare:

- post coinvolto;
- data;
- severita;
- stato: `open`, `needs_fix`, `closed`, `superseded`;
- evidenza consultata;
- azione richiesta;
- data e motivo di chiusura, quando risolto.

## Registro Audit

|---|---|---|---|---|
| `2026-07-07-post-visual-audit.md` | multipli | da rivedere | storico | aggiungere sezione risoluzioni confermate |
